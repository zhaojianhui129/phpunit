# phpunit
phpunit练习笔记

###安装
####下载
```sh
$ curl -fSL "https://phar.phpunit.de/phpunit.phar" -o "phpunit.phar"
```
或者
```sh
$ wget https://phar.phpunit.de/phpunit.phar
$ php phpunit.phar --version
```

全局安装
```sh
$ wget https://phar.phpunit.de/phpunit.phar
$ chmod +x phpunit.phar
$ sudo mv phpunit.phar /usr/local/bin/phpunit
$ phpunit --version
```
